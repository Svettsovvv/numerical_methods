Документация к программе task_one

1. Назначение программы
Программа решает систему линейных алгебраических уравнений (СЛАУ) вида AX = B, где:
- A — матрица коэффициентов размерности 20x20,
- B — вектор правой части размерности 20,
- X — вектор неизвестных, который требуется найти.

2. Основные алгоритмы и функции

a. Шаги программы:
a_1. Считывание данных из файлов:
   - Матрица A считывается из файла 'RGEmtr094.dat'.
   - Вектор B считывается из файла 'RGErhs094.dat'.

a_2. Модификация матрицы A и вектора B:
   - Для каждого значения параметра P:
     - К элементу A(1,1) добавляется текущее значение P(k).
     - К элементу B(1) добавляется 8 * P(k).

a_3. Решение системы:
   - Для каждого измененного набора A и B вызывается подпрограмма gaussian_method, которая решает систему методом Гаусса.

a_4. Вывод результатов:
   - Для каждого значения P выводится решение системы X.

b. Подпрограмма gaussian_method
- Алгоритм:
  b_1. Прямой ход метода Гаусса:
     - Приведение матрицы A к верхнетреугольному виду.
     - Преобразование вектора B в соответствии с преобразованиями матрицы.
     - Шаги:
       - Для каждого k от 1 до n-1:
         - Для каждой строки i от k+1 до n:
           - Вычисление множителя factor = A(i, k) / A(k, k).
           - Вычитание из строки i строки k, умноженной на factor.
           - Обновление элемента B(i) = B(i) - factor * B(k).

  b_2. Обратный ход метода Гаусса:
     - Нахождение значений вектора X, начиная с последнего уравнения.
     - Шаги:
       - Вычисление X(n) = B(n) / A(n, n).
       - Для каждого i от n-1 до 1:
         - Вычисление X(i) = (B(i) - сумма A(i, j) * X(j) для j от i+1 до n) / A(i, i).
